About the Project

This is a site that recommends an anime randomly, in addition to having an API that can be used and is consumed by the website itself in combination with the Kitsu API to obtain external resources.

This site is presented as a project of the distributed systems area, created by Valeria Arias and Duvan Ibarra.

J2Logo

Website where we found the explanation and got the idea for our project.

Kitsu

Website from which we obtained images and some extra data about the anime.

Bootstrap

Framework used for the frontend of our website.

Flask

Framework used to implement the backend of our site, including the API.

Python

The programing language used to put all together what is related to backend.
