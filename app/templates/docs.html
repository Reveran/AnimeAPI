API Usage

{{url}}api/

You can make a GET request to this url to obtain a random anime every time you make it.

{{url}}api/id

Make a Get request replacing id with the id number of a knowing anime to obtain information of an anime, for example:
"{{url}}1" will return you "Bleach's information"
